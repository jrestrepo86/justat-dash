## Descripción

Implementación en Plotly-Dash de una vista del tablero JUSTAT incorporando información gráfica.

## Instalación

```bash
git clone https://github.com/jrestrepo86/justat-dash.git
cd justat-dash
venv activate
pip install -r requirements.txt
python app.py
```
