## Instalación (virtual env)

```bash
git clone https://github.com/jrestrepo86/justat-dash.git
cd justat-dash/app
python -m venv env
source env/bin/activate
pip install -r requirements.txt
python3 app.py
```

## Instalación (docker)

```bash
git clone https://github.com/jrestrepo86/justat-dash.git
cd justat-dash
docker compose up
```
